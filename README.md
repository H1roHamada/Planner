## Используемые технологии

### Основное:

- React v18
- NextJS v14
- TanStack Query v5

### Дополнительно:

- React Hook Form
- DnD-kit
- DayJS

### UI:

- AntD
- Tailwind

### Backend:

- NestJS v10
- Prisma v5
- PostgreSQL
